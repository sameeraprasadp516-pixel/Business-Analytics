This project presents a Business Analytics Dashboard developed using Power BI to analyze employee and organizational data. The dashboard provides meaningful insights into salary distribution, employee performance, and attrition trends.

🔍 Project Overview

The dashboard focuses on:

Salary distribution based on experience
Relationship between performance score and salary
Attrition analysis (Yes/No)
Department-wise salary comparison
Employee performance vs salary trends
Age and salary insights
📈 Key Features
Interactive salary range filter
KPI cards for quick summary (e.g., total experience)
Multiple visualizations (bar charts, line charts, pie chart)
Clean and professional layout for easy understanding
🛠 Tools & Technologies
Power BI
Microsoft Excel
🎯 Purpose

This project is created as a beginner-level analytics work to demonstrate:

Data visualization skills
Dashboard design techniques
Basic HR/business data analysis
📌 Output

The dashboard helps in understanding employee performance patterns, salary structure, and attrition trends, which can support better decision-making in organizations.
